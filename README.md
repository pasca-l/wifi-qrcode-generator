# Wifi QRcode Generator
QRcode generator for Wifi information.

## Screenshots
![](https://github.com/user-attachments/assets/b00c3164-ddb1-4728-a933-a02ab14708d5)

## Requirements
- Docker 27.3.1
- Docker Compose v2.29.7

## Usage
1. Start server.
```shell
$ docker compose up
```

1. Open website (port set to 8080).
```
http://localhost:8080
```
